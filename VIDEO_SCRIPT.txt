ACTION: Scroll to Test 2 code-gen output.

SAY:

==========================================================
  SECTION 5  |  TEST 3 — IF-ELSE  |  0:50 – 1:05
==========================================================

ACTION: Scroll to Test 3 code-gen output.

SAY:
  "Test 3 is an if-else statement.
   The condition  x > 5  is evaluated into a temporary t0.
   The code generator emits:
     IF_FALSE t0 GOTO L0  — jump to the else block if false,
     GOTO L1              — skip the else if true,
     L0: label for the else branch,
     L1: merge point after both branches.
   This is exactly how real compilers implement control flow."
